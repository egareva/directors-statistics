Добавление файла
1. Название файла совпадает с названием канала
2. Первая строчка в файле КАНАЛ: название_канала
3. Вторая строчка в файле ССЫЛКА: ссылка_на_канал
4. Фильмы должны быть разделены точкой с запятой
5. Список фильмов должен быть в скобках
6. Если у фильма указан год, он должен быть в скобках

Проверка после добавления
1. Запустить проверку check_similarity.py
2. Первая проверка: на количество не больше 30 и одинаковых режиссеров в скиске
3. Имена режиссеров проверяются на схожесть написания
4. Если есть похожее, заменить на того, кто в кинопоиске
5. При замене написать комментарий, начиная с #
6. Названия фильмов внутри каждого режиссера проверяются на схожесть

Дополнительная проверка
1. Запустить проверку list_all_names.py
2. Выведется список режиссеров по алфавиту, проверить похожесть
3. Выведется список режиссеров и количество упоминаний (от большего к меньшему)
4. Выведется список по рейтингу
5. Выведется список режиссеров с уникальными фильмами (если есть несколько одинаковых фильмов, их надо унифицировать)
